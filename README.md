# Titanic Dataset EDA

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset.

## Objectives
- Analyze survival rate by gender, class, and age
- Visualize important patterns
- Generate insights from data

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Females had higher survival rates
- 1st class passengers survived more
- Children had better survival chances
